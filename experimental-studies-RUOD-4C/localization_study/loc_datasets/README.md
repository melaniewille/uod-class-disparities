 ## RUOD-4C localization data
 Please download the [**localization datasets for RUOD-4C**]() from Hugging Face and place them here.
 The folder structure should look like this:
 ```
 localization_study/
 ├── loc_datasets/
 │   ├── single_class_data/
 │   ├── balanced_data/
 │   └── reduced_data/
 └── ...
 ```
